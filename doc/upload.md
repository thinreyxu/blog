在 Node.js 中使用文件上传
==========================

根据[nswbmw][1]的[说法][2]，在 Node.js 中有三种实现文件上传的方法：
1. 使用 Express 自带的文件上传功能
2. 使用 Formidable 外部模块
3. 使用 MongoDB 的 GridFS 功能
其中，前两种不涉及数据库，而第三种需要使用数据库
